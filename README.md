# Trust-Aware Hybrid Phishing Detection

## 📖 Overview

A hybrid Explainable Artificial Intelligence (XAI) framework for phishing detection that integrates machine learning, deep learning, trust-aware decision making, domain reputation analysis, and adversarial robustness testing to improve the reliability and transparency of phishing detection.

The framework integrates a Random Forest model for URL analysis and a Bidirectional Long Short-Term Memory (BiLSTM) model for email content analysis. Predictions from multiple components are combined using a hybrid decision mechanism enhanced with trust scoring and explainability techniques to provide transparent and reliable phishing detection.

The project follows a modular 10-phase research workflow covering data preparation, model development, hybrid fusion, explainability, trust assessment, security analysis, adversarial evaluation, and deployment through an interactive Gradio-based web interface.
